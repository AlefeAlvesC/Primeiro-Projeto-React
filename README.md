# Projeto React - Introdução com Vite

## 📌 Sobre o Projeto
Este projeto foi desenvolvido como uma introdução ao uso do **React** para criação de interfaces web baseadas em componentes. Durante o desenvolvimento, foram explorados conceitos básicos do ecossistema React e ferramentas modernas utilizadas no desenvolvimento front-end.

O projeto tem como objetivo compreender a estrutura inicial de uma aplicação React, a criação de componentes e a utilização de ferramentas de build modernas.

---

## ⚙️ Tecnologias Utilizadas
- **React**
- **Vite**
- **Node.js / npm**
- **JavaScript**
- **HTML / CSS**

---

## 🚀 O que foi praticado

### 1. Criação e utilização de componentes
Foi criado um componente React simples para entender o conceito de **componentização**, que é um dos princípios fundamentais do React.  
Os componentes permitem dividir a interface em partes reutilizáveis, facilitando a organização e manutenção do código.


---

### 2. Uso de `export default`
Foi utilizado o **`export default`** para exportar componentes e permitir sua importação em outros arquivos do projeto.

---

### 3. Primeira experiência com Vite e npm

O projeto foi criado utilizando Vite, uma ferramenta moderna de build que oferece:

- Inicialização rápida do projeto
- Atualização instantânea no navegador (Hot Reload)
- Ambiente de desenvolvimento mais leve

O npm foi utilizado para instalar dependências e executar scripts do projeto.

---

## 🎯 Objetivo do Projeto

Este projeto teve como objetivo:
- Entender a estrutura básica de um projeto React
- Criar e utilizar componentes
- Aprender a exportar e importar componentes
- Ter o primeiro contato com Vite e npm no desenvolvimento front-end

## 📚 Aprendizados

Durante o desenvolvimento foi possível compreender melhor:
- A organização de projetos em React
- A importância da componentização
- O funcionamento básico do fluxo de desenvolvimento com Vite
- A utilização do npm para gerenciamento de dependências

✏️ Projeto desenvolvido para fins de estudo e prática em React.
